CFG Simplification

A context-free grammar can always be simplified without changing the language it generates. The process has four ordered steps — skipping or reordering them can break correctness.

01

Eliminate ε-productions — rules of the form A → ε

02

Remove unit productions — rules of the form A → B (single variable)

03

Remove non-generating symbols — can never derive a terminal string

04

Remove unreachable symbols — never appear in a derivation from S

💡 The order matters: ε first, then unit, then useless. Wrong order leaves behind symbols that should have been removed.

ε-Production Elimination

An ε-production is any rule A → ε. The goal is to remove all such rules while preserving every non-empty string the grammar generates.

1

Find all nullable variables: X where X ⇒* ε. Start with direct ε-rules, then close transitively.

2

For each production, generate every subset by optionally removing nullable symbols from the right-hand side.

3

Delete all original ε-productions. Keep S → ε only if ε belongs to the language.

Example: If B is nullable and you have A → a B c, add A → a c as a new rule.

Unit Production Removal

A unit production is A → B where B is a single variable. These must be removed after ε-elimination.

1

Compute unit pairs: (A, B) where A ⇒* B using only unit steps. Close transitively from (A, A) for each A.

2

For each unit pair (A, B), copy all non-unit productions of B into A's production set.

3

Remove all original unit productions from the grammar.

Example: If S → A → B → a | b, then S and A both get a | b added, and all unit rules are deleted.

Useless Symbol Removal

A symbol is useless if it can never generate a terminal string (non-generating) or can never be reached from S (unreachable).

→

Non-generating: no derivation from X ever produces only terminals. Remove these first.

↔

Unreachable: X never appears in any sentential form derived from S. Remove after non-generating pass.

⚠ Always remove non-generating before unreachable. The wrong order can leave behind symbols that appear reachable but are still useless.

Correct Order

These four steps must be applied in this exact sequence for a correct result:

01

ε-elimination — removing unit rules before this causes new ε-productions to go unnoticed.

02

Unit production removal — all ε-rules are gone, so unit pairs are clean to compute.

03

Remove non-generating — rules are clean enough to check generation accurately.

04

Remove unreachable — final pass to strip anything S can no longer reach.

After all four steps, the grammar is in reduced form — every variable is reachable, generating, and free of unit/ε rules.

☕ Format Guide

One production per line. Use → or -> to separate left and right sides.

Use | for multiple alternatives. Use ε or eps for the empty string.

Variables = uppercase (A–Z)  ·  Terminals = lowercase/symbols

S → A B | ε
A → a A | a
B → b B | ε
C → c ← removed if unreachable

💡 Use the ε and → buttons above the input to insert symbols without typing them.
